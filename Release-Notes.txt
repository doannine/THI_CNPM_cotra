**2. Chú giải làm thế nào để chạy Source Code**

**Yêu cầu hệ thống:**
- Hệ điều hành: Windows.
- Ngôn ngữ: Nodejs 
- Cài đặt các package yêu cầu trong file `package.json`.
- Database: MySQL 
**Hướng dẫn chi tiết:**
-Cài node js vào máy
-khởi tạo  sql  
-Đã cài sẵn các package trong file package.json chỉ cần git clone code về máy và npm start chạy ứng dụng
